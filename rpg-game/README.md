# Shadow Chronicles — 어둠의 연대기

브라우저에서 바로 실행되는 다크 판타지 RPG 게임입니다.

## 실행 방법

`index.html` 파일을 브라우저에서 열면 바로 실행됩니다. 별도 설치나 서버 설정이 필요 없습니다.

## 파일 구조

```
rpg-game/
├── index.html   # 게임 화면 전체 HTML 구조
├── style.css    # 스타일 및 애니메이션
└── game.js      # 게임 로직 전체 (데이터, 시스템, UI)
```

## 주요 기능

### 계정 시스템
- 회원가입 / 로그인 (localStorage에 계정 저장)
- 게임 저장 및 불러오기 (계정별 세이브 데이터)

### 직업 (클래스)
| 직업 | 특징 | 대표 스킬 |
|------|------|-----------|
| 전사 | 고 HP, 고 방어력 | 강타, 방패막기, 전투의 함성 |
| 마법사 | 고 마법 공격, 저 HP | 화염구, 번개화살, 메테오 |
| 도적 | 고 속도, 고 치명타율 | 기습, 독 단검, 은신 |

### 세계 지도
- WASD 또는 방향키로 캐릭터 이동
- E키 또는 클릭으로 장소 입장
- 레벨 제한이 있는 던전은 잠금 표시

| 장소 | 필요 레벨 | 종류 |
|------|-----------|------|
| 출발의 마을 | 1 | 마을 |
| 어둠의 동굴 | 1 | 던전 |
| 저주받은 숲 | 4 | 던전 |
| 폐허의 성채 | 6 | 던전 |
| 불의 화산 | 8 | 던전 |
| 용의 둥지 | 12 | 던전 (보스) |

### 던전 이벤트
던전 탐험 중 랜덤으로 다음 이벤트가 발생합니다.
- 전투 (45%) — 몬스터 조우
- 보물 (20%) — 아이템 및 골드 획득
- 함정 (15%) — HP 피해
- 휴식 (10%) — HP 회복
- 아무것도 없음 (10%)

### 전투 시스템
- 턴제 전투 (플레이어 선공)
- 일반 공격, 스킬, 아이템 사용 가능
- 상태이상: 독, 빙결, 기절, 은신, 방패막기
- 치명타 시스템 (클래스별 치명타율 상이)

### 마을 시설
- **여관** — 20골드로 HP/MP 완전 회복
- **상점** — 아이템 구매 및 판매
- **대장간** — NPC 대화
- **모험가 길드** — NPC 대화

## 게임 데이터 저장

모든 데이터는 브라우저의 `localStorage`에 저장됩니다.

- 계정 정보: `shadowChroniclesAccounts` 키
- 게임 세이브: 계정 데이터 내 `saveData` 필드
